# WhatsApp-Web-Message-Spam-Tool
A fun and experimental Chrome extension that spams messages in the currently open WhatsApp Web chat. Made for entertainment and testing purposes only.

## What Can It Do?

- Spam the same message multiple times
- Set how many messages to send
- Set delay between messages
- Add random IDs to bypass simple spam detection
- Only spams the **currently open chat**
